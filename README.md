# cmd-stack
This app aims to replace the disorganized, hard-to-search notepad files that programmers use to store their notes. I created this app to help programmers quickly save, organize and search their personal collection of commands and tidbits of notes. One of the big principles was speed of searching and adding new commands. This is why this is a CLI tool first and GUI second. The backend is written in Rust and uses ReDB as the database. The frontend is an Electron app which is powered by TypeScript.
